# Algorithm-practice-C

C언어로 작성하는 알고리즘 및 자료구조 연습 저장소

---

## 📁 폴더 구조

```
Algorithm-practice-C/
├── C-language/          # C 기본 개념 연습 (printf, 포인터, 메모리 등)
├── Data-Structure/      # 자료구조 구현 (연결 리스트, 스택, 큐, 트리 등)
├── Baekjoon/            # 백준 온라인 저지 풀이
│   ├── sorting/
│   ├── search/
│   ├── dp/
│   └── ...
└── Programmers/         # 프로그래머스 풀이
    ├── sorting/
    ├── search/
    └── ...
```
